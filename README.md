# Task-Manager
Description
A simple Task Manager web application that allows users to add, complete, 
and delete their daily tasks. Tasks update instantly on the page without any
reload, keeping the experience smooth and straightforward.


Technology Used: - 
1.HTML
2.CSS
3.JavaScript



Steps: -
1.Type a task in the input box
2.Click Add Task or press Enter to add it
3.Check the checkbox to mark a task as completed
4.Click Delete to remove a task from the list
